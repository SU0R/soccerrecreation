Algonquin Titans Soccer

Controls

A browser-playable version of the original Python/Pygame soccer game. The game opens full-screen in the Pygame web runtime so the keyboard controls work cleanly.

Play Game

Blue

Q/W/E select players, X steals, F/S/D passes, A/Z keeper.

Red

I/O/P select players, L steals, M/J/K passes, H/N keeper.

Ball

V starts a shot, then click the target. R restarts the match.
