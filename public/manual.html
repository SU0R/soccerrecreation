Instruction Manual

This is a browser-deployable version of the original Python/Pygame soccer project. It keeps the same idea: select a player, click where they should move, steal when close to the ball, pass between teammates, and shoot on goal.

Blue Team

  • Q: select B1
  • W: select B2
  • E: select B3
  • X: steal when close to the ball
  • F / S / D: pass to B1 / B2 / B3
  • A / Z: move the blue keeper up / down

Red Team

  • I: select R1
  • O: select R2
  • P: select R3
  • L: steal when close to the ball
  • M / J / K: pass to R1 / R2 / R3
  • H / N: move the red keeper up / down

Ball + Match

  • Click: send the selected player to that location
  • V: start a shot, then click the target
  • R: restart the match
  • Score by sending the ball into the opposite goal.
  • The match timer runs for 180 seconds.

Online Multiplayer Roadmap

The game code now keeps most important match data inside one state object. That makes a future room-based multiplayer version cleaner: clients can send inputs, a shared authority can validate them, and the browser can render the same state. Vercel can host the site, but the realtime socket layer should use a provider such as Pusher, Ably, Supabase Realtime, Liveblocks, or PartyKit.
